# Alphabet Bahasa Isyarat

![Alphabet bahasa isyarat](https://i.pinimg.com/originals/5d/38/ed/5d38ed4f4ed71134c84c19fcd2a6aaa0.jpg)

Referensi di atas menampilkan alfabet manual bahasa isyarat huruf A–Z. Chart seperti ini sangat berguna sebagai acuan visual saat membentuk huruf dengan tangan, terutama untuk percobaan gesture recognition dan visualisasi interaktif.

> Catatan: referensi ini mengikuti gaya alfabet manual berbasis ASL. Jika kamu memakai standar bahasa isyarat lokal tertentu, beberapa bentuk bisa berbeda.

## Cara membaca chart

- **A–I, K–Y** biasanya ditampilkan sebagai bentuk tangan statis.
- **J dan Z** umumnya dibentuk dengan gerakan, bukan posisi diam.
- Satu tangan mewakili satu huruf.

## Referensi huruf

| Huruf | Bentuk / cara membuat |
| --- | --- |
| A | Kepalkan tangan, ibu jari berada di sisi luar. |
| B | Telapak menghadap keluar, empat jari rapat dan lurus, ibu jari menempel di telapak. |
| C | Bentuk tangan melengkung seperti huruf C. |
| D | Telunjuk lurus ke atas, jari lain menyentuh ibu jari membentuk lingkaran. |
| E | Jari-jari menekuk ke dalam, ibu jari menekan bagian depan jari. |
| F | Ibu jari dan telunjuk membentuk lingkaran kecil, tiga jari lain lurus ke atas. |
| G | Ibu jari dan telunjuk sejajar, menunjuk ke samping seperti ukuran kecil. |
| H | Telunjuk dan jari tengah lurus sejajar ke samping. |
| I | Jari kelingking lurus ke atas, jari lain mengepal. |
| J | Bentuk seperti I, lalu gerakkan tangan membentuk huruf J di udara. |
| K | Telunjuk dan jari tengah lurus membentuk huruf V, ibu jari menahan jari manis. |
| L | Telunjuk dan ibu jari membentuk huruf L. |
| M | Tiga jari menutup ibu jari. |
| N | Dua jari menutup ibu jari. |
| O | Semua jari membentuk lingkaran oval seperti huruf O. |
| P | Seperti K tetapi mengarah ke bawah. |
| Q | Seperti G tetapi mengarah ke bawah. |
| R | Telunjuk dan jari tengah disilangkan. |
| S | Kepalan tangan penuh. |
| T | Ibu jari masuk di antara telunjuk dan jari tengah yang menutup. |
| U | Telunjuk dan jari tengah lurus rapat ke atas. |
| V | Telunjuk dan jari tengah membentuk huruf V. |
| W | Tiga jari lurus ke atas membentuk huruf W. |
| X | Telunjuk ditekuk seperti kait. |
| Y | Ibu jari dan kelingking terbuka, jari lain mengepal seperti huruf Y. |
| Z | Seperti angka 1 dengan telunjuk, lalu gerakkan tangan membentuk huruf Z di udara. |

## Ringkasan cepat

Kalau ingin menghafal dengan cepat:

- **A / S / T / E / M / N / O** = variasi kepalan dan posisi ibu jari
- **B / C / D / G / H / I** = posisi jari yang lebih terbuka
- **J / Z** = huruf bergerak
- **K / L / V / W / Y** = bentuk jari yang paling mudah dibedakan secara visual

## Relevansi untuk project

Referensi alfabet ini digunakan untuk:

- mengenali gesture huruf
- menguji translasi gesture ke teks
- menciptakan visualisasi partikel yang berubah mengikuti bahasa isyarat
